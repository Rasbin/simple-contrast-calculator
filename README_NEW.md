# Color Contrast Calculator

A professional tool to ensure color combinations meet WCAG 2.1 accessibility standards.

## What is it?

The Color Contrast Calculator helps designers and developers verify that text and background color combinations have sufficient contrast ratios for accessibility compliance. It supports WCAG Level AA (4.5:1) and AAA (7:1) standards.

## How it works

1. **Set Colors** - Enter or adjust three colors using hex values or HSL sliders
2. **View Current Ratios** - See the current contrast ratio between each color pair and their WCAG compliance status
3. **Define Requirements** - Specify which contrast ratios (AA or AAA) you need for each color pair
4. **Find Solutions** - Select which color to adjust and the tool suggests accessible alternatives that meet your requirements
5. **Preview Changes** - View side-by-side previews of how suggested colors work together

## Quick Start

```bash
npm start        # Run development server
npm run build    # Create production build
```
